# 3. Observability

DuploCloud offers two levels of observability: **Standard** **Observability** and the **Advanced Observability Suite (AOS)**.

* **Standard Observability** provides basic monitoring for Services and Containers, including logs, metrics, and cloud spend.
* **Advanced Observability Suite (AOS)** is an add-on that delivers comprehensive, dynamic observability across all resources and Tenants, with enhanced features such as traces, profiling, alerts, and deeper analytics.

# Databases

DuploCloud supports the following databases for Microsoft Azure. Use the procedures in this section to set them up.

* [SQL Database](../../../automation-platform/overview-2/azure-services/databases/sql-database.md) (MSSQL)
* [PostgreSQL](../../../automation-platform/overview-2/azure-services/databases/postgresql.md)
* [PostgreSQL Flexible Server](../../../automation-platform/overview-2/azure-services/databases/postgresql-flexible-server.md)
* [MySQL Server](../../../automation-platform/overview-2/azure-services/databases/mysql-server-database/)
* [MySQL Flexible Server](../../../automation-platform/overview-2/azure-services/databases/mysql-flexible-server.md)
* [Redis](../../../automation-platform/overview-2/azure-services/databases/redis-database.md)
* [CosmosDB](../../../automation-platform/overview-2/azure-services/databases/cosmosdb.md)

{% hint style="info" %}
When using Azure Databases, select a Tenant other than the **Default** Tenant.
{% endhint %}

# Databases

DuploCloud supports the following databases for AWS. Use the procedures in this section to set them up.

* [RDS](../../../../aws-user-guide/aws-services/database/rds-database/) (MariaDB, MySQL, PostgreSQL, SQL-Express, SQL-Web, SQL-Standard, Aurora MySQL, Aurora MySQL Serverless, Aurora PostgreSQL, Aurora PostgreSQL Serverless)
* [DynamoDB](../../../../aws-user-guide/aws-services/database/dynamodb.md)
* [Timestream](timestream-database.md)
* [ElastiCache](elastic-cache.md) (Redis, Memcached)
